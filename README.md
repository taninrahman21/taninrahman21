## 🧑‍💻 About Me

I'm a **Software Developer from Dhaka, Bangladesh** specializing in **WordPress plugin development**, **Gutenberg blocks**, and **React-based interfaces**. For the past 2+ years I've been a core developer at [**bPlugins**](https://bplugins.com/), helping build and grow their plugin products from the ground up.

- 🔌 Core developer on **7 WordPress plugins** that together reached **20,000+ active installs** at bPlugins
- ⚡ Built, shipped, and maintained features across the full plugin lifecycle — from block API to admin UI to support
- 🤝 Handled user support, wrote documentation, and fixed production bugs for real-world users
- 🌍 Open to **remote opportunities** globally
- 📖 Currently learning **TypeScript**, **Next.js**, and **AI integrations**
- 💬 Ask me about **WordPress**, **Gutenberg block API**, **React**, or **plugin architecture**
